    # MyBatis-Plus

    # 1、介绍

    MP是MyBatis的增强工具，在MyBatis的基础上只做增强不做改变，为简化开发、提高效率。只需要简单配置，即可实现CRUD

    # 7、乐观锁

    应用场景：当要更新一条记录的时候，希望这条记录没有被别人更新，就是实现线程安全的更新

    实现方法：取出记录时，获取当前version，更新时，带上这个version。执行更新时，set version = newVersion where version = oldVersion,如果version不对时，就更新失败。应用场景：抢羽毛球场

    1. 在表中添加字段作为版本号，在表对应的实体类添加版本号属性version，并且在类中添加属性

    2. 在属性上面添加注解

          @Version
    private Integer version;

    4. 逻辑删除

      • 物理删除：真实删除，将数据库中表中记录删除了
      • 逻辑删除：假删除，将对应数据中代表是否被删除字段修改为被删除状态，但是在数据库表中还能够被查询到，但是查询的时候查不到

      使用方法：

      • 在表中添加字段，作为逻辑删除标志，每次删除时候，修改标志位。0 - 没有删除，1-删除

      1. 在数据库中添加字段，对应实体类添加属性，作为逻辑删除标志位

      2. 在实体类对应属性添加注解

            //表示为删除标志位
    @TableLogic
    //添加初始值
    @TableField(fill = FieldFill.INSERT)
    private  Integer delete;
